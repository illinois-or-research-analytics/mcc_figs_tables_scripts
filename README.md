# mcc_figs_tables_scripts
Supporting code used to generate figures and tables in "modeling citations and cartels" manuscript submitted as a preprint (multiple versions) to Zenodo
under concept DOI: 10.5281/zenodo.19368703. The original version of the preprint is also available on Research Square under DOI: 10.21203/rs.3.rs-9270520/v1. The Zenodo version will be kept up to date. 

### data\_generation
The folder named data generation contains configuration files and scripts used to generate the raw data that was processed by the R and Python scripts in this repository. The directory structure is preserved and matches the expected structure by the supporting code. Mentions of prefixes such as /projects/illinois/eng/cs/chackoge/illinoiscomputes/abm/v6/abm\_outputs should be interpreted as the data\_generation folder of this repository.

Cartel inputs, such as the subfolder labelled\_node\_edge\_list, were generated by first running the simulation using the configuration file and scripts in the corresponding folder followed by running the Python script generate\_cartels.py, included in the data\_generation folder.
